# Project Management Backend

A scalable backend system for managing projects, tasks, and collaboration with role-based access control.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features
- User authentication (JWT, email verification, password reset)
- Role-based access control (Admin, Project Admin, Member)
- Project, task, subtask, and notes management
- File upload and validation
