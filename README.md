# 🚀 Project Management Backend

A scalable backend system for managing projects, tasks, and team collaboration with secure authentication and role-based access control.

---

## 🧰 Tech Stack
- **Node.js**
- **Express.js**
- **MongoDB**
- **JWT Authentication**
- **REST APIs**

---

## ✨ Features
- 🔐 User authentication (JWT-based login & signup)  
- 📧 Email verification & password reset workflows  
- 👥 Role-based access control (Admin, Project Admin, Member)  
- 📁 Project, task, subtask, and notes management  
- 📤 Secure file upload and validation  
- ⚙️ Structured API design with proper error handling  

---

## 🏗️ Architecture
- Modular backend structure  
- RESTful API design  
- Scalable and maintainable codebase

## ⚙️ Setup Instructions

### 1. Clone the repository
```
git clone https://github.com/harshit-404/project-management-backend.git
```

### 2. Install dependencies
```
npm install
```

### 3. Create a `.env` file
```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### 4. Run the server
```
npm start
```
